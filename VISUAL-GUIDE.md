## Эффекты и анимации

### Transitions:
```css
--transition-fast:   0.2s (hover на ссылки)
--transition-normal: 0.3s (кнопки, карточки)
--transition-slow:   0.5s (большие изменения)
```

### Animations:
- `fadeInDown` — логотип hero
- `fadeInUp` — заголовки, таймер, кнопки
- `Scroll reveal` — карточки при прокрутке

### Hover эффекты:
- Карточки: `translateY(-8px)` + фиолетовые тени
- Кнопки: `translateY(-2px)` + увеличенные тени
- Ссылки: подчёркивание снизу
- Галерея: `scale(1.05)` + overlay


## Рекомендации по контенту

### Фотографии:

**Качество:**
- Разрешение: Full HD (1920x1080) или выше
- Формат: JPG (сжатый до 80-85% качества)
- Или: WebP для лучшей производительности

**Стиль:**
- Профессиональные фото
- Хорошее освещение
- Фокус на танцорах
- Динамика и движение

**Размеры:**
- Hero background: 1920x1080 (не нужен, используется логотип!)
- Stats cards: 800x800
- Program cards: 800x1000
- Format images: 1000x600
- Jury photos: 600x600
- Gallery: 800-1200px

### Текст:

**Заголовки:**
- Короткие и ёмкие
- Без точек в конце
- Крупный шрифт

**Описания:**
- Понятные и конкретные
- Без "воды"
- Польза для участника
