# История изменений — танец.фест

## Версия 2.1 (29 января 2026)

### Изменения дизайна

#### Цветовая схема
**Было:** Тёмная тема (черный + красный акцент)  
**Стало:** Светло-фиолетовая тема (как в арт-портале)

**Обновлённые цвета:**
- Основной фон: `#ffffff` (белый)
- Градиентный фон body: светло-фиолетовый градиент
- Акцент: `#9c27b0` → `#ab47bc` (фиолетовый)
- Текст: `#2d2d2d` (тёмный) вместо белого
- Карточки: белый фон с фиолетовыми тенями

#### Hero Section
**Главное изменение:** Логотип из `logo/logofest.png` теперь используется как фоновое изображение на весь экран

**Было:**
```css
background: темный градиент + placeholder фото
```

**Стало:**
```css
background: светло-фиолетовый градиент + logo/logofest.png
```

**Эффект:** Красивый цветной логотип видим на фоне Hero секции

#### Компоненты

**Header:**
- Фон: белый с прозрачностью
- Текст: фиолетовый
- Бордер: светлый

**Кнопки:**
- Primary: фиолетовый (#9c27b0)
- Hover: тёмно-фиолетовый (#7b1fa2)
- Тени: фиолетовые

**Таймер:**
- Фон карточек: белый с прозрачностью
- Цифры: тёмно-фиолетовый
- Бордер: фиолетовый с прозрачностью
- Тени: фиолетовые

**Форма:**
- Фон inputs: светло-серый (#fafafa)
- Focus: фиолетовый бордер
- Загрузка файла: фиолетовые акценты

**Footer:**
- Фон: тёмно-фиолетовый (#7b1fa2)
- Текст: белый
- Hover: белый

#### Секции
- **stats:** белый фон
- **program:** светло-фиолетовый (#faf8fc)
- **future-formats:** белый
- **jury:** светло-фиолетовый
- **about:** белый
- **gallery:** светло-фиолетовый
- **application:** градиент (белый → светло-фиолетовый)

### Технические изменения

**CSS Variables обновлены:**
```css
--primary-light: #f3e5f5
--primary: #ce93d8
--primary-medium: #ba68c8
--primary-dark: #9c27b0
--primary-darker: #7b1fa2
--accent: #ab47bc
```

**Фоновое изображение Hero:**
```css
background: 
    linear-gradient(135deg, rgba(243, 229, 245, 0.85), rgba(206, 147, 216, 0.4), rgba(186, 104, 200, 0.3)),
    url('logo/logofest.png') center/cover no-repeat;
```

**Градиент body:**
```css
background: linear-gradient(135deg, #f3e5f5 0%, #e1bee7 50%, #ce93d8 100%);
background-attachment: fixed;
```

### Удалённые элементы
- Убран отдельный блок `<div class="hero-logo">` из HTML
- Удалены стили `.hero-logo` и `.hero-logo-img` из CSS

### Сохранено
- Вся структура сайта
- Все функции (таймер, форма, меню)
- Адаптивность
- Логотипы в header и footer
