## Выполненные задачи

### 1. ✅ Hero-секция: увеличение размера и улучшение градиента

**Что было сделано:**
- Увеличена высота Hero с `min-height: 100vh` до `110vh`
- Добавлен верхний отступ `140px` вместо `var(--spacing-3xl)`, чтобы логотип не наезжал на header
- Изменен размер логотипа на фоне: `background-size: 80%` и `center/contain` вместо `cover`
- Усилен градиент для лучшей читаемости текста:
  - Было: `rgba(0.25, 0.2, 0.15)` - слишком прозрачно
  - Стало: `rgba(0.6, 0.5, 0.45)` - текст хорошо читается
- Добавлен дополнительный градиент через `::after` для улучшения контраста

**Результат:** Логотип виден на фоне, не наезжает на header, текст отлично читается.

- В HTML добавлена 6-я карточка "100+ номинаций" вместо широкой карточки с призовым фондом

**Результат:** 6 одинаковых карточек в 2 ряда по 3, весь текст виден полностью.

### 4. ✅ Видео: 2 карточки с модальным окном

**Что было сделано:**

#### HTML:
- Заменен единственный `.video-placeholder` на `.video-grid` с 2 карточками
- Каждая карточка имеет `onclick="openVideoModal(url)"` для открытия модального окна
- Добавлено модальное окно `#videoModal` с iframe для воспроизведения видео
- Кнопка закрытия модального окна

#### CSS:
- Создана сетка `.video-grid` с 2 колонками
- Стилизованы `.video-card` с hover-эффектами
- Создано модальное окно `.video-modal` с темным фоном и центрированным контентом
- Кнопка закрытия с анимацией и hover-эффектом
- Адаптивность: на мобильных видео в 1 колонку

#### JavaScript:
- Функция `openVideoModal(videoUrl)` - открывает модальное окно и запускает видео
- Функция `closeVideoModal()` - закрывает окно и останавливает видео
- Закрытие по клавише Escape
- Функции сделаны глобальными через `window`

**Результат:** 2 видео-карточки рядом, при клике открывается модальное окно с воспроизведением.
