## Что было исправлено

### Проблема
Логотип `logofest.png` не отображался в Hero-секции из-за двух проблем:
1. Неправильный путь к файлу в CSS
2. Слишком плотный градиент, перекрывающий изображение

### Решение

В файле `css/style.css` (строки 358-368) внесены изменения:

#### 1. Исправлен путь к логотипу
```css
/* Было */
url('logo/logofest.png')

/* Стало */
url('../logo/logofest.png')
```
CSS файл находится в папке `css/`, поэтому нужен относительный путь с `../`

#### 2. Уменьшена прозрачность градиента
```css
/* Было */
linear-gradient(135deg, 
    rgba(243, 229, 245, 0.85) 0%, 
    rgba(206, 147, 216, 0.4) 50%, 
    rgba(186, 104, 200, 0.3) 100%)

/* Стало */
linear-gradient(135deg, 
    rgba(243, 229, 245, 0.25) 0%, 
    rgba(206, 147, 216, 0.2) 50%, 
    rgba(186, 104, 200, 0.15) 100%)
```
Прозрачность снижена с 85%/40%/30% до 25%/20%/15%
