## Что изменилось

### Цветовая схема

**Переход:** Тёмная тема → Светло-фиолетовая (как в арт-портале)

**Основные цвета:**
```css
--primary-light: #f3e5f5   /* Светло-фиолетовый */
--primary: #ce93d8          /* Средний фиолетовый */
--primary-dark: #9c27b0     /* Фиолетовый */
--primary-darker: #7b1fa2   /* Тёмно-фиолетовый */
--accent: #ab47bc           /* Акцент */
```

**Фон:**
- Body: светло-фиолетовый градиент (fixed)
- Секции чередуются: белый ↔ светло-фиолетовый
- Карточки: белый фон с фиолетовыми тенями

### Hero Section — главное изменение!

**Фоновое изображение:**
Логотип `logo/logofest.png` теперь на фоне Hero секции на весь экран!

```css
background: 
    linear-gradient(светло-фиолетовый с прозрачностью),
    url('logo/logofest.png') center/cover;
```

**Эффект:**
- Красивый цветной логотип фестиваля виден на фоне
- Поверх него: название, таймер, кнопки
- Градиент создаёт глубину и читаемость текста
