# Итоговый отчёт: танец.фест 2.0

## Проект завершён!

Создан полностью обновлённый сайт фестиваля "танец.фест" — Всероссийский фестиваль хореографического искусства.

---

## Что было сделано

### Полное обновление дизайна

**Концепция:**
- Современный элегантный тёмный дизайн
- Вдохновлён арт-порталом (визуальная иерархия, сетка, типографика)
- Адаптирован под тематику танцевального фестиваля
- **Без эмодзи и декоративных иконок** — только фотографии и типографика

**Цветовая схема:**
- Тёмная палитра (черный, серый)
- Акцентный цвет: красный (#ff3366)
- Минималистичный и элегантный

### Структура сайта (8 основных секций)

#### 1. Hero Section (Главный экран)
- Логотип фестиваля из папки `logo/`
- Крупный заголовок: "танец.фест 2.0"
- Подзаголовок и дата: "16 мая 2026"
- **Таймер обратного отсчёта** (JavaScript)
  - Дни, часы, минуты, секунды
  - Автоматическое обновление каждую секунду
- CTA кнопки: "Принять участие" и "Программа"

#### 2. Ключевые показатели
Оформлено карточками с фоновыми фотографиями:
- 1000+ участников
- 40+ хореографов
- 2000+ зрителей
- Профессиональное жюри
- Денежный призовой фонд + подарки

**Без иконок!** Только крупные цифры поверх качественных фотографий.

#### 3. Программа фестиваля
Танцевальные направления в формате карточек с фото:
- Детская хореография (до 7 лет)
- Современная хореография
- Стилизованная хореография
- Народная хореография

#### 4. Будущие форматы
Три карточки с фотографиями и описанием:
- Мастер-классы от топовых хореографов
- Шоукейсы (командные выступления)
- Афтерпати (DJ-сеты и свободный танец)

#### 5. Жюри фестиваля
6 карточек с портретными фотографиями:
- Имя и фамилия
- Звание / статус
- Краткое описание (направление, достижения)

**Стиль арт-портала:** чистые карточки, минимализм, акцент на фотографии.

#### 6. О фестивале
- Текстовый блок с описанием
- Преимущества участия (01, 02, 03, 04)
- Галерея атмосферных фотографий
- Статус и масштаб события

#### 7. Прошедшие фестивали
- Фото-галерея с адаптивной сеткой
- Видео с прошлого фестиваля (placeholder с кнопкой Play)
- Hover-эффекты на изображениях

#### 8. Форма подачи заявки
Полнофункциональная форма с:
- Имя и фамилия
- Email
- Телефон (автоформатирование)
- Категория участия
- Коллектив / Солист
- Дополнительная информация
- **Загрузка MP3 файла** (до 50 МБ)
- Валидация всех полей
- Отправка на email

## Как запустить

### Быстрый способ (PHP):
1. Откройте `submit.php`, измените email (строка 22)
2. Загрузите файлы на хостинг
3. Откройте в браузере

### Node.js способ:
```bash
npm install
cp env.example .env
# Заполните .env
npm start
# http://localhost:3000
```

## Изображения (важно!)

Сайт использует placeholder изображения от Unsplash для демонстрации.

**Перед запуском замените на реальные фотографии:**

1. **Hero background** — танцоры в движении (1920x1080)
2. **Статистика** — выступления (800x800)
3. **Программа** — по направлениям (800x1000)
4. **Жюри** — портретные фото (600x600)
5. **Галерея** — атмосферные фото (800-1200px)

**Как заменить:**
В `index.html` найдите:
```html
style="background-image: url('https://images.unsplash.com/...');"
```

Замените на:
```html
style="background-image: url('images/your-photo.jpg');"
```
