# танец.фест 2.0 — Всероссийский фестиваль хореографического искусства

Современный сайт танцевального фестиваля с таймером обратного отсчёта, адаптивным дизайном и формой подачи заявок.

## Особенности

### Дизайн
- Элегантный тёмный дизайн, вдохновлённый арт-порталом
- Акцент на фотографии и типографику
- Без эмодзи и декоративных иконок
- Современная визуальная иерархия
- Полная адаптивность (mobile / tablet / desktop)

### Функциональность
- **Таймер обратного отсчёта** до даты фестиваля
- Плавная анимация и scroll-эффекты
- Фиксированная навигация
- Мобильное бургер-меню
- Форма заявки с валидацией
- Загрузка MP3 файлов
- Отправка на email

## Быстрый старт

### Вариант 1: PHP

1. **Настройте email в `submit.php`**
   ```php
   // Строка 22
   define('RECIPIENT_EMAIL', 'your-email@example.com');
   ```

2. **Загрузите файлы на хостинг**
   ```
   Через FTP загрузите:
   - index.html
   - css/
   - js/
   - logo/
   - submit.php
   ```

3. **Готово!** Откройте сайт в браузере

### Вариант 2: Node.js

1. **Установите зависимости**
   ```bash
   npm install
   ```

2. **Настройте `.env`**
   ```bash
   cp env.example .env
   nano .env
   ```

3. **Запустите сервер**
   ```bash
   npm start
   # Откройте http://localhost:3000
   ```

## Решение проблем

### Таймер не работает
- Проверьте правильность даты в `js/main.js`
- Убедитесь, что JavaScript загружается
- Откройте консоль браузера (F12)

### Форма не отправляется
- Проверьте `submit.php` или `server.js`
- Проверьте SMTP настройки
- Откройте консоль браузера для ошибок

### Изображения не загружаются
- Замените placeholder URL на реальные пути
- Проверьте права доступа к файлам
- Проверьте правильность путей

### Адаптив не работает
- Очистите кэш браузера
- Проверьте viewport meta тег
- Проверьте media queries в CSS
